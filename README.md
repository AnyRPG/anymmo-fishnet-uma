# AnyMMO FishNet UMA Addon

## Introduction

This is the AnyMMO FishNet UMA addon for AnyRPG.

It adds the following features and functionality:
* UMA male and female characters configured for use with FishNet
* Template content that includes FishNet UMA NPCs

## Installation

This addon can be installed via the Welcome Window in AnyRPG.

https://github.com/AnyRPG/AnyRPGCore
